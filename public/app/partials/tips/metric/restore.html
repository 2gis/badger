"Restore data" - search JIRA issues for period

Example: find all issues for metric since 2015/06/01 with step = 1 week

Metric using in example: total count of created issues in the "CD" project

1. In the field "Jira filter for period" type jql query for search issues (paying attention on dates)

project = CD and created >= "2015/06/01"

2. In "Handler field" select field to group issues by

created   // cause we group created issues

3. In "Step (in days)" type the step to group issues

7         // cause we group with step = 1 week

4. Grouped issues are handled by function "Total count"

If you don't want to find issues for period, leave all fields empty.
