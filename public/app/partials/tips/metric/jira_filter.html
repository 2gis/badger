"Jira filter" - JQL query to search for JIRA issues

Example 1: find all created issues in the "CD" project in the last week

project = CD and created >= "-1w"

Example 2: find all closed issues in the "CD" project in the last week

project = CD and status changed to Closed after "-1w"

Please call your attention to date format.

Better to use values as "-1d", "-1w", "startOfMonth()" (depends on cron-template).

I.e. for metrics, collected once a week, you should use "-1w".

Example 3: find all backlog issues in the "CD" project

project = CD and status="Backlog"

Date is not needed.
